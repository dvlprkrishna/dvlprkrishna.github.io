Hi, I'm Stupid Blocker.

A not so smart, site blocker.

What is stupid blocker?

A work in progess site blocker. Currently dumb but, will soon be updated with many smart features.
tl:dr - It allows you to block sites like facebook, youtube and other sites that you are addicted too.

## Installation

The easiet way to install this plugin - Download stupidblocker.crx - file from here.
Drag and drop the file into Google chrome browser.


Go to chrome://extensions/ check developer mode > Load unpacked extension. > select "stupidblocker" folder > done.

# Current Features

- Redirects user away from blocked websites.
- Hard to modify or delete url
- No toggle options
- Hard to edit redirect url

# Upcoming Features

- Easy way to add url like a popup input
- Easy way to modify or delete url
- A way to toggle extension on and off
- Custom redirect page url input

## Development

By default stupid blocker blocks facebook, youtube and quora.com for adding more sites to block list a user has to edit files.

# Manifest.json

- To add more sites to block list open "manifest.json" file in any notepad and edit "https://www.facebook.com/*"
- you can add more urls or modify existing but make sure the urls follows this format "https://www.sitenamehere.com/*" , "https://www.site2namehere.com/*" , "https://www.site3namehere.com/*"

# Basic.js

- you can even change the page this extension redirects to, by editing the "basic.js" window.location.replace("Insert Url here");
