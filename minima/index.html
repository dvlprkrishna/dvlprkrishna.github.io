About

Hi, We're Minima


Minima is a bootstrap theme designed and developed with minimalism in mind. Unlike other minimalist design out there, we tried to keep things dead simple. There are two color scheme used for minima, light and dark version.


Minima is easy to customize based on individual requirements, font size, color, etc are easy to change. We have included theme documentation and some ready made pages as well. Like Pricing, FAQ, 404 error page to help you get started.

Features

Responsive Design

Minima is developed using bootstrap framework with reponsiveness and mobile first approach. which make our site look great on desktop and mobile devices.

Lighting Fast

Minima make use of latest coding standard, code reusablity. We make use of CDN to serve framework and javascript files. Which reduces site load time and display website faster.

Multipurpose Design

Minima has a multipurpose design for use in various field. Like web design studio, portfolios, small startups, and many others.

Absolutely Free

Minima is free of cost. There are no hidden charges, no monthly subscription, no credit card required. 100% free.
